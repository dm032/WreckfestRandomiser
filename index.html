Wreckfest Randomiser

For creating random races for Wreckfest lobbies. Turn tracks on and off using the settings, then generate a random list of races, demolition, or a mix. Use the NEXT TRACK button to load the next track in your selected list (if you want to avoid repeats), or continue using the random buttons for a fully random experience.
